# Concept: GIT Fundamentals

## What it is
Git is the actual version control system while git hub is a website that hosts your repository. Git is free and open source.
> Linux and Mac OS normally have git already installed. 

## How it works
Git commands:
Push
Pull
Clone
Commit
Status
checkout
reset
## Real-world example
teams use git to track changes in code, and keep a history to roll back to if needed as well as lets multiple programers work on a project.
